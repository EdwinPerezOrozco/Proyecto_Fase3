    Información General

    El presente OVI es realizado con el fin de dar a conocer informacion y es netamente educativo, con el fin de que el estudiante obtenga conocimientos basicos sobre Git, Git Hub, HTML5 y CSS3, por otra parte encontrara actividades que ayudaran a la practica de lo aprendido en la plataforma.

    Objetivos del OVI

    El objetivo del OVI es enseñar al estudiante el manejo de las herramientas HTML5, CSS3 y GIT HUB, en donde por medio de las diferentes fuentes de informacion se lograra un aprendizaje basico y continuo. Por otra parte en el presente OVI hay material de multimedia y actividades integradas para poner en practica lo aprendido.

    Objetivos del Curso

    Exponer y dar a conocer el proceso necesario para diseñar un sitio web mediante las distintas herramientas y/o lenguajes de programación existentes, facilitando así el acceso y comprensión de la información por parte de la persona interesada

    Tematicas

    Github:Es una plataforma de desarrollo colaborativo de software para alojar proyectos usando el sistema de control de versiones Git.

    HTML5: Es la última versión de HTML. El término representa dos conceptos diferentes: Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos.

    CSS: Es un lenguaje para definir el estilo o la apariencia de las páginas web, escritas con HTML o de los documentos XML.
